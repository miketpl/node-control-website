Everything you need to manage your network

From quick show commands to full API exploration, Node Control covers your daily ops workflow.

🔌

Connection Manager

SSH into devices with saved credentials, stored securely in your system keychain. Tabbed sessions, reconnect handling, and credential management built in.

📡

API Explorer

Browse and execute API calls across Meraki, Cisco DNA Center, ISE ERS, and PAN-OS. 42+ preset operations with custom path support and JSON editing.

🗺

Network Topology

Visualise your network with auto-discovered L2/L3 topology maps. CDP/LLDP neighbours, MAC address tables, ARP, and routing tables — all interactive.

💻

Interactive Terminal

Full SSH terminal with Tab completion, command history, and PAN-OS scripting mode. Works like your favourite terminal emulator, inside the app.

💾

Config Backup

One-click configuration backups across your device fleet. Schedule backups, compare diffs, and track configuration changes over time.

📋

Task Runner

Execute bulk operations across multiple devices. File transfers, packet captures, and custom command sequences — with real-time progress tracking.

Safe Mode — production confidence

Choose your risk level at startup. Safe Mode enforces read-only access across the entire app.

🔒

Safe Mode

Read-only access. Show commands only — no config changes. Terminal mode and HTTPS disabled. Perfect for monitoring production networks.

⚙️

Config Mode

Full access. All commands, terminal mode, HTTPS sessions, API mutations, and tasks enabled. Use with caution on production.
