Powerful Features

Everything you need to manage your network infrastructure from a single desktop application.

Network Discovery

Automatically discover devices across your network using CDP, LLDP, and SNMP. Build a complete inventory in minutes.

L2/L3/WAN Topology

Interactive topology maps for Layer 2, Layer 3, and WAN with live VLAN, STP, MAC table, and route table flyouts with search.

Configuration Management

Push configurations, run commands, and manage device settings across your entire fleet from one interface.

Automated Backups

Schedule automatic configuration backups with diff comparison. Never lose a config change again.

Real-time Monitoring

Live interface status, bandwidth monitoring, ping, and traceroute — all integrated into the topology view.

Device Library Sync

GitHub-backed device library keeps your team in sync. Share device definitions and templates across installations.

See It In Action

A modern, dark-themed interface designed for network engineers who live in the terminal.
