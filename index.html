Everything Automated

One desktop app. Every network operation. AI-powered intelligence baked in. Built for engineers who live in the terminal but need visibility across the whole infrastructure.

Network Discovery

Drop in a seed IP and watch your network map itself. CDP, LLDP, and SNMP discovery builds a complete inventory in minutes — no spreadsheets required.

L2 / L3 / WAN Topology

Interactive topology maps with live VLAN, STP, MAC table, and route table flyouts. Right-click any node for instant SSH, ping, traceroute, or table lookups with full search.

Push Config at Scale

Broadcast commands to your entire fleet or target individual devices. Safe mode, config mode, dry-run — your network, your rules.

Automated Backups & Tasks

Scheduled config backups with diff comparison, automated tasks from the task runner, and repeatable workflows — so you always know what changed, when, and by whom.

Terminal Modes

Connect to one device or hundreds at once. Command mode, full interactive terminal, or HTTPS via embedded browser — with safe mode to stop you breaking anything.

API Explorer

Built-in API explorer for Meraki, DNA Center, PAN-OS, and Cisco ISE. Execute calls, view JSON responses, and provision devices — no Postman needed.

Nine specialist AI agents that understand your network. Ask questions in plain English — get topology analysis, config recommendations, what-if scenarios, and root cause diagnosis.
